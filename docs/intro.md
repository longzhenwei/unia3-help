# 前言

## **简介**

友霓UNIA3平台是目前市场首家实现企业与企业间交互的SaaS云管理平台。有别于传统SaaS ERP只能实现各企业独立在云端使用，UNIA3中每个企业在平台中都是独立的， 企业间的业务可以交互完成，依托完备的数据交互逻辑，为企业提供了更多机遇。
